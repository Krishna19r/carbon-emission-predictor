## Regression-Based Carbon Emission Prediction from Cloud Resource Usage

##  Project Overview
With the rapid adoption of cloud computing, even small web applications contribute to carbon emissions due to energy-intensive data centers. This project aims to build a regression-based machine learning model that predicts CO₂ emissions based on cloud resource usage.

---

## Objectives
- Predict carbon emissions from cloud usage
- Analyze impact of infrastructure parameters
- Promote sustainable cloud computing
- Build an end-to-end ML deployment pipeline

---

##  Planned Architecture

Cloud Usage Data  
→ Preprocessing  
→ Feature Engineering  
→ Regression Model  
→ CO₂ Prediction  
→ Dashboard Visualization  

---

##  Features

### Input Parameters:
- CPU Usage (%)
- Memory Usage (GB)
- Storage Usage (GB)
- Requests per Day
- Uptime Hours
- Data Transfer (GB)
- Carbon Intensity
- PUE (Power Usage Effectiveness)

### Target:
- CO₂ Emissions (kg per month)
